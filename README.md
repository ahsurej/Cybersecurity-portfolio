# Cybersecurity Learning Portfolio

#

I am a Bachelor of Information Technology student with a strong interest in Cybersecurity, particularly in Security Operations (SOC), Digital Forensics, and Penetration Testing.

This portfolio documents my learning journey as I build practical skills in cybersecurity through hands-on labs, projects, hackathons, and self-study.

## Purpose of This Portfolio

The goal of this repository is to:

* Track my progress in cybersecurity
* Demonstrate practical, hands-on learning
* Build a strong foundation for future specialization in SOC and related fields
* Showcase my readiness for internships, certifications, and graduate studies

## Current Focus Areas

* Networking fundamentals (TCP/IP, DNS, HTTP/HTTPS)
* Linux and Windows security basics
* Security Operations (SOC) concepts
* Introductory penetration testing
* Basic digital forensics and log analysis

## Tools & Platforms Used

* Wireshark
* TryHackMe
* VirtualBox / Virtual Machines
* Linux (Ubuntu)
* Windows Event Viewer
* GitHub for documentation

## Portfolio Structure

Each folder in this repository represents a different area of learning:

* **Networking/** → Traffic analysis, protocols, Wireshark labs
* **Linux/** → Commands, permissions, system exploration
* **SOC/** → Log analysis, incident response exercises
* **TryHackMe/** → Completed rooms and write-ups
* **Hackathons/** → Project work and contributions
* **Projects/** → Independent cybersecurity experiments

Each project contains:

* A README explaining the objective and process
* Screenshots of work done
* Notes and findings
* Any supporting files (if applicable)
